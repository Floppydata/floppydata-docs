## Base URL

```text
https://client-api.floppy.host
```

All endpoints in this reference are under the `/v1` prefix.

## Authentication

Authenticate requests with your API key in the `X-Api-Key` header.

```bash
curl --request GET \
  --url 'https://client-api.floppy.host/v1/rotating/balance' \
  --header 'X-Api-Key: YOUR_API_KEY'
```

You can manage API keys in [account settings](https://app.floppydata.com/settings/account).
