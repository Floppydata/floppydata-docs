> Send your first Web Unlocker request in minutes.

You can start directly in the interface or generate code and use it in your app right away.

<Note>
  Every account includes **5 free successful requests**.
</Note>

## Two ways to start

<CardGroup cols={2}>
  <Card title="Try it in the interface">
    Paste a URL, click **Scrape**, and review the result instantly.
  </Card>

  <Card title="Use it in code">
    Create an API key, copy the generated request, and run it in your workflow.
  </Card>
</CardGroup>

<Tip>
  Both options use the same request logic — the interface is simply the fastest way to test it.
</Tip>

## Step-by-step setup

<Steps>
  <Step title="Open Web Unlocker">
    Go to your dashboard and open [Web Unlocker](https://app.floppydata.com/tools/scrape).
  </Step>
  <Step title="Enter a URL">
    Paste any public webpage into the input field.

    <video src="/videos/0414(1).mp4" autoPlay loop muted playsInline />

    _You can also use presets for supported websites to get started faster._\
    \
    **Try it in the interface** - Click **Scrape** to send the request and review the result in Output preview.

    <video src="/videos/0414(2).mp4" autoPlay loop muted playsInline />
  </Step>
  <Step title="Create an API key">
    Click **Get code**, then create or select your **API key**. Once selected, the key is added to the generated request automatically.

    <video src="/videos/0414(3).mp4" autoPlay loop muted playsInline />

    ### Example

    ```bash
    curl https://client-api.floppy.host/v1/webUnlocker \
      --request POST \
      --header 'Content-Type: application/json' \
      --header 'X-Api-Key: YOUR_API_KEY' \
      --data '{
        "url": "https://floppydata.com/",
        "country": "US",
        "city": "New York",
        "difficulty": "low",
        "expiration": 0
      }'
    ```
    Copy the generated code and use it in your terminal, backend, or scripts.
  </Step>
</Steps>
