## Overview

The Floppydata Client API lets you work with:

- Rotating proxy traffic and locations
- Static IP inventory
- Web Unlocker requests
- Proxy validation tools

Base URL: `https://client-api.floppy.host`

## Quick Request

```bash
curl --request GET \
  --url 'https://client-api.floppy.host/v1/rotating/balance' \
  --header 'X-Api-Key: YOUR_API_KEY'
```

<Note>
  Use the `X-Api-Key` header for all requests.
</Note>
