<Note>
  Floppydata offers **five proxy types** grouped into **two categories**:

  - **Rotating proxies** – IP changes across requests or sessions
  - **Static proxies** – IP remains fixed

  Each type differs in trust level, speed, cost, and ideal use cases.
</Note>

## Rotating proxies

Rotating proxies automatically assign IPs from a shared pool.

<Note>
  Rotating proxies are billed based on **traffic (GB)**, not individual IPs.
</Note>

### Residential

- real user IPs from ISPs
- high trust level
- large global pool

> **Best for:** scraping, SEO, geo-targeting, protected websites

### Mobile

- IPs from mobile carrier networks
- highest trust level
- shared across real mobile users

> **Best for:** social media, multi-accounting, high-risk targets

### Datacenter

- server-based IPs
- fast and cost-efficient
- easier to detect

> **Best for:** bulk scraping, non-protected targets

## Static proxies

Static proxies provide a fixed IP.

<Note>
  Static proxies are billed **per IP per month**.
</Note>

### Static ISP

- real ISP IPs (residential-like)
- stable identity
- high trust

> **Best for:** accounts, long sessions, SEO tools

### Static Datacenter

- fixed server IP
- fast and stable
- lower trust

> **Best for:** automation, infrastructure, whitelisting
