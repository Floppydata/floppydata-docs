> Extract data from any website with a single request — no proxy setup, no scraping infrastructure.

## What it is

Web Unlocker is a scraping API that returns clean, ready-to-use webpage data.

Instead of managing proxies, headers, and anti-bot logic yourself, you just send a request with a URL — and get the result.

<Note>
  You don’t need to configure proxies, rotation, or headers manually.\
  **Web Unlocker** handles everything automatically.
</Note>

## How it works

<CardGroup cols={2}>
  <Card title="Send a request">
    You provide a target URL and request the page through Web Unlocker.
  </Card>

  <Card title="Automatic unblocking">
    Floppydata selects the best IP, adjusts headers, bypasses protections, and retries if needed.
  </Card>

  <Card title="Clean response">
    You get HTML or structured output ready for parsing.
  </Card>

  <Card title="No manual setup">
    No proxy rotation, anti-bot tuning, or retry logic required on your side.
  </Card>
</CardGroup>

<Tip>
  In most cases, a simple request with just a URL is enough to get a valid result.
</Tip>

## What you can do

Use Web Unlocker to:

- scrape product pages, search results, and listings
- access geo-restricted content
- collect data from protected websites
- run large-scale scraping without managing infrastructure

## When to use it

| Use case | Web Unlocker fit |
| --- | --- |
| Protected websites | Excellent |
| Large-scale scraping | Strong |
| Geo-restricted content | Strong |
| Simple websites with no blocking | Optional |
| Full browser automation / account sessions | Not the best fit |

<Note>
  If your use case doesn’t require full unblocking logic, regular proxies may be enough.
</Note>
