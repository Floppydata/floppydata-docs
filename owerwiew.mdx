**Floppydata is a proxy infrastructure platform for controlled, scalable access to web resources.**

Instead of managing proxies manually, you describe how a request should behave – and Floppydata handles routing automatically.

## Two core products

### Proxies

Control how your requests appear to target websites. Route traffic through residential, mobile, ISP, or datacenter IPs with precise geo-targeting and session control.

**Best for**: Web scraping, SEO monitoring, account management, automation, when you need control over request parameters.

### Web Unlocker

Automatic anti-bot bypass API. Send a URL, get clean HTML back. No proxy configuration, no header tuning, no retry logic needed.

**Best for**: Protected websites (Cloudflare, PerimeterX), large-scale scraping, when you just need data without managing infrastructure.

\
**Proxy vs Web Unlocker: Which to use?**

| Your need | Use Proxy | Use Web Unlocker |
| --- | --- | --- |
| Simple websites, no anti-bot | ✔ | Optional |
| Protected sites (Cloudflare, etc) | ✗ | ✔ |
| Full control over requests | ✔ | ✗ |
| Browser automation (Selenium, Puppeteer) | ✔ | ✗ |
| Just need clean data, no complexity | ✗ | ✔ |
| Custom headers, cookies, logic | ✔ | Limited |
| Geo-targeting required | ✔ | ✔ |
| Session persistence needed | ✔ | ✗ |
| High-volume, protected targets | ✗ | ✔ |

<Tip>
  **Still not sure?** Try both:

  - Start with **Proxy** if you're integrating with existing code or tools
  - Start with **Web Unlocker** if you're scraping protected sites
</Tip>
