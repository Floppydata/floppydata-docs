## 1. Get API Key

Generate or copy your API key in [account settings](https://app.floppydata.com/settings/account).

## 2. Export Environment Variables

```bash
export FLOPPY_API_KEY='YOUR_API_KEY'
export FLOPPY_BASE_URL='https://client-api.floppy.host'
```

## 3. Make Your First Calls

Check traffic balance:

```bash
curl --request GET \
  --url "$FLOPPY_BASE_URL/v1/rotating/balance" \
  --header "X-Api-Key: $FLOPPY_API_KEY"
```

List available countries and cities:

```bash
curl --request GET \
  --url "$FLOPPY_BASE_URL/v1/rotating/locations" \
  --header "X-Api-Key: $FLOPPY_API_KEY"
```

Validate a proxy:

```bash
curl --request POST \
  --url "$FLOPPY_BASE_URL/v1/tools/checkProxy" \
  --header "Content-Type: application/json" \
  --header "X-Api-Key: $FLOPPY_API_KEY" \
  --data '{
    "connectionString": "http://username:password@geo.g-w.info:10080"
  }'
```
