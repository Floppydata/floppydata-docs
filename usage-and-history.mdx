> Track your requests, monitor performance, and debug issues in real time.

Web Unlocker provides a built-in usage dashboard where you can review all requests, their status, and response details.

## What you can see

<CardGroup cols={2}>

  <Card title="Status & results">

  </Card>

  <Card title="Usage tracking">
    Monitor how many requests you’ve used and how many remain.
  </Card>

  <Card title="Debugging">
    Inspect request details to understand failures or unexpected results.
  </Card>
</CardGroup>
